  1. 2021-09-17
  2. 111字
  3. 1 min read

记一次docker-compose 转发错误

log如下：

WARNING: IPv4 forwarding is disabled. Networking will not work.

大概是IPv4转发禁用，解决办法如下:

系统:centos 7-8.x(64位)

1.修改内核参数

vim /etc/sysctl.conf
net.ipv4.ip_forward=0
改成
net.ipv4.ip_forward=1

2.重启network服务

systemctl restart network

3.重启docker服务

systemctl restart docker

记一次docker卸载有残留的问题

sudo apt remove --purge  docker docker-ce docker-compose-plugin docker-desktop docker-scan-plugin docker.io
